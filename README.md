# Hu Tao "Kasih Tau Mama" Ver Code

Anime-style motion animation inspired by the trending song **"Kasih Tau Mama"**.

This project was built using **ReactJS** and **GSAP** to create smooth transitions, motion effects, and anime-style visual editing directly in the browser.

---

## ✨ Features

* Smooth anime motion transitions
* GSAP timeline animations
* Text sync effects
* Browser-based animation rendering
* Responsive fullscreen layout

---

## 🛠 Tech Stack

* ReactJS
* GSAP
* JavaScript
* CSS

---

## 📦 Installation

Clone this repository:

```bash
git clone https://github.com/Difarft2/hutao-kasih-tau-mama.git
```

Open project folder:

```bash
cd hutao-kasih-tau-mama
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
